# ACLS-timer

This repository contains the ACLS Timer HTML page.

Files:

- `ACLS-timer.html` — the web page you can publish with GitHub Pages. Open it directly at `./ACLS-timer.html` after cloning, or at the Pages URL after publishing.

Publishing with GitHub Pages:

1. Go to the repository Settings → Pages.
2. Set the source to the `main` branch and the root (/) folder.
3. After the site builds, the page will be available at: `https://allanoyster.github.io/ACLS-timer/ACLS-timer.html`
